### 一、相关技术

- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`、
- 缓存：`Redis`

### 二、开发软件（列出的软件其一均可运行）

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`

### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`
- 缓存：`Redis3.0`

### 四、功能描述及功能图展示

系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 新闻动态、4. 推荐新闻、5. 浏览记录、6.个人信息

管理员功能：1. 登录、2. 用户管理、3. 管理员管理、4. 新闻管理、5. 知识图谱
